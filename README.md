# RPGgamev2 (Tentative Name)

This project is a recreation of a game that was made as a homework assignment that I made during a coding bootcamp back in 2019. It is still a work in progress when complete it should play similar to games like Fire emblem or Final Fantasy Tatics. This project uses BablyonJS, React, and typescript.
